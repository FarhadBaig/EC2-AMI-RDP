# AWS Windows VM Task

## Objective
Create a Windows EC2 instance on AWS and connect using RDP.

## Steps Performed
1. Created Windows EC2 instance.
2. Configured RDP access.
3. Retrieved Administrator password.
4. Connected using Remote Desktop.
5. Opened Command Prompt.
6. Executed `systeminfo`.
7. Viewed Windows About System information.
